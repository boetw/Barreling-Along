#BARRELING ALONG
 Boet Waldbjorn
## What it is
 Javascript game using phaser framework.


## How to Play
 Use left and right keys to steer, dodging rocks as long as possible.
 Game switches between two players and keeps track of scores as you go.


##Technologies

* HTML
* CSS
* JavaScript
* Phaser
